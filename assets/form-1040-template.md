# Form 1040 Output Template Reference

Use this as a checklist when verifying the generated form output.

## Required Fields Checklist

### Personal Info
- [ ] Taxpayer name
- [ ] SSN (last 4 only)
- [ ] Filing status
- [ ] Spouse name (if MFJ)
- [ ] Dependent count

### Income (must have at least one source)
- [ ] Wages (Line 1z)
- [ ] Interest (Line 2b)
- [ ] Dividends (Line 3b)
- [ ] Capital gains (Line 7)

### Deductions
- [ ] Standard vs itemized decision documented
- [ ] QBI deduction if business income present

### Tax
- [ ] Ordinary income tax
- [ ] SE tax if Schedule C present
- [ ] NIIT if investment income + AGI exceeds threshold

### Credits
- [ ] CTC if children under 17
- [ ] EITC if income within range

### Payments
- [ ] Federal withholding from all W-2s
- [ ] Estimated payments if applicable

### Result
- [ ] Refund or amount owed
- [ ] Effective tax rate
- [ ] Marginal rate

### Disclaimer
- [ ] "AI-generated estimate" notice included
- [ ] "Review with CPA" recommendation
- [ ] Big Beautiful Bill caveat if provisions applied
